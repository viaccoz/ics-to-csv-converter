# ICS → CSV Converter

A web-based tool to convert `.ics` calendar files (or ZIPs containing multiple `.ics`) into a CSV file.

## Features

- Upload single `.ics` or ZIP of `.ics` files
- Parse events including:
  - Calendar name (`X-WR-CALNAME`)
  - Summary
  - Start date
  - End date
  - Location
  - Recurrence
  - Description
- Sortable table
- Search/filter events in the table
- Download all events as CSV
